Explain “repaint” and “reflow” of the webpage.

Repaint is when the look but not layout of a webpage changes, needing a visual update. Reflow is when the layout changes, requiring the whole page to be recalculated and redrawn.

How does JavaScript on the webpage affects the page load speed and performance?

JavaScript can slow down a webpage if it’s large or used inefficiently, but techniques like async loading help.

What is “minification” of the script? Why it is useful to minify your scripts?

Minification makes JavaScript files smaller by removing extra characters, which speeds up page loading.

What is a JS Web Worker and when do we need it?

A Web Worker lets JavaScript run in the background without slowing down the webpage, useful for heavy tasks.
